02Manejo de Errores y Excepciones

Como lograr que tu programa no se caiga ante un dato incorrecto o una situacion inesperada.

❌ Error de Sintaxis

El codigo esta mal escrito y Python ni siquiera puede ejecutarlo. Por ejemplo, olvidar los dos puntos de un if. Se corrigen reescribiendo el codigo; no se pueden "manejar".

⚠️ Excepcion

El codigo esta bien escrito y empieza a correr, pero falla mientras se ejecuta por culpa de los datos o el entorno. Estas SI se pueden manejar con try/except.

Excepciones mas comunes

ExcepcionCuando ocurreEjemplo
ValueErrorEl valor tiene el tipo correcto pero no sirveint("abc")
TypeErrorSe opera con tipos incompatibles"5" + 3
ZeroDivisionErrorDivision (o modulo) por cero10 / 0
IndexErrorIndice fuera de rango de una lista[1,2][5]
KeyErrorClave inexistente en diccionariod["x"]
NameErrorVariable que no fue definidaprint(total)
FileNotFoundErrorArchivo que no existeopen("x.txt")

📋 pip: Comandos esenciales

$ pip install numpy # Instalar la ultima version
$ pip install numpy==1.26.4 # Version especifica
$ pip list # Ver librerias instaladas
$ pip show numpy # Info de una libreria
$ pip uninstall numpy # Desinstalar
$ pip freeze > requirements.txt # Guardar dependencias
$ pip install -r requirements.txt # Instalar desde archivo

🌐 Entornos Virtuales

Una carpeta aislada con las librerias propias de cada proyecto. Evita conflictos entre versiones.

$ python -m venv venv # 1. Crear entorno virtual
$ venv\Scripts\activate # 2. Activar (Windows)
$ source venv/bin/activate # 2. Activar (Linux/Mac)
$ pip install numpy # 3. Instalar (queda solo aqui)
$ deactivate # 4. Salir del entorno
Idea clave: Un entorno virtual por proyecto + un requirements.txt = tu proyecto se puede instalar y ejecutar en cualquier computador sin sorpresas. Es la forma profesional de trabajar.

04Auditoria de Codigo con SonarQube

Como revisar automaticamente la calidad de tu codigo mientras lo escribes, dentro de Visual Studio Code.

🔍 Analisis Estatico

Examina el codigo fuente sin ejecutarlo, buscando problemas conocidos. Es como un corrector ortografico pero para programacion.

🔒 Confiabilidad

Posibles bugs que harian fallar el programa.

🛡️ Seguridad

Vulnerabilidades que un atacante podria aprovechar.

📋 Mantenibilidad

"Code smells": codigo que funciona pero es dificil de leer o mantener.

Instalacion en VS Code

  1. Abre el panel de Extensiones: icono de bloques en la barra lateral, o Ctrl+Shift+X
  2. Busca SonarQube
  3. Elige SonarQube for IDE (autor: Sonar) y pulsa Install
  4. Espera a que termine. No requiere configuracion para analizar archivos Python locales.

Nota: Antes se llamaba "SonarLint". Es la misma herramienta, renombrada.

Tipos de hallazgos

TipoSignificado
● BugError que probablemente hara que el programa funcione mal
● VulnerabilityProblema de seguridad que podria ser explotado
● Security HotspotCodigo sensible que conviene revisar manualmente
● Code SmellCodigo que funciona pero dificulta la lectura o mantenimiento

05Ejercicios Propuestos

Practica los cuatro temas con problemas simples. Todos incluyen validacion de datos.

BASICO

01 Calculadora de operaciones basicas

Pide al usuario dos numeros y un operador (+, -, *, /) y muestra el resultado.

  • Validar que ambos numeros sean numericos (manejar ValueError)
  • Si el operador es division y el segundo numero es 0, avisar sin que el programa se caiga

Conceptos: operadores, input(), try/except, ZeroDivisionError

BASICO

02 Validar un numero en un rango

Pide una nota entre 1.0 y 7.0 y no avances hasta que el usuario ingrese un valor valido.

  • Repetir con while hasta obtener un dato correcto
  • Rechazar texto y numeros fuera del rango 1.0 - 7.0
  • Indicar si es aprobatoria (>= 4.0) o reprobatoria

Conceptos: while True, float(), validacion de rango, condicionales

INTERMEDIO

03 Promedio de notas con listas

Pregunta cuantas notas se ingresaran, pide cada una, guardalas en una lista y calcula promedio, maxima y minima.

  • Validar cada nota antes de agregarla (reutiliza el patron del ejercicio 2)
  • Usar append() para llenar la lista
  • Calcular con sum(), len(), max() y min()

Conceptos: listas, append(), sum/len/max/min, ciclo for

BASICO

04 Conversor de temperatura

Pide una temperatura en grados Celsius y conviertela a Fahrenheit y Kelvin.

  • Formulas: F = C * 9/5 + 32 y K = C + 273.15
  • Validar que la entrada sea un numero
  • Mostrar resultados redondeados a 2 decimales

Conceptos: operaciones algebraicas, float(), round(), validacion

INTERMEDIO

05 Resolver una ecuacion cuadratica

Resuelve a*x^2 + b*x + c = 0 pidiendo los coeficientes a, b y c. Usa la libreria math.

  • Calcular discriminante d = b^2 - 4*a*c
  • Usar math.sqrt(d) solo si d >= 0
  • Validar que a no sea 0

Conceptos: libreria math, math.sqrt, condicionales, validacion

INTERMEDIO

06 Agenda telefonica con diccionario

Crea una mini agenda con menu: (1) agregar, (2) buscar, (3) listar, (4) salir.

  • Usar un ciclo while que repita el menu
  • Al buscar, manejar nombre inexistente con get() o capturando KeyError
  • Para listar, recorrer con items()

Conceptos: diccionarios, get()/items(), menu con while, KeyError

BASICO

07 Palabras unicas con conjuntos

Pide una frase y muestra cuantas palabras distintas tiene, ignorando repeticiones.

  • Separar la frase con frase.split()
  • Guardar en un set para quedarte solo con las unicas
  • Mostrar cantidad con len() y el conjunto resultante

Conceptos: conjuntos (set), split(), len(), datos unicos
